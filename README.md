# 3D-Raycasting-Engine-for-Architectural-Design
While architects traditionally design buildings using 2D blueprints, visualising the final structure in 3D can be time-consuming and difficult. Therefore, this Python program explores converting 2D architectural blueprints into a real-time 3D virtual walkthrough tool; Built from scratch, no external code, and minimal set of libraries (ex:Pygame).
